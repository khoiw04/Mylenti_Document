Want to embed Bilibili videos in your blog pages?  
Here’s a step-by-step guide to develop a simple Bilibili video component.

---

<script setup>
import BiliVideo from '../components/BiliVideo.vue'
</script>

:::details Video Component Example
<BiliVideo bvid="BV1q3411b7ps"/>
<br>
:::

## Implementation

Create a component named `BiliVideo` inside your `components` folder, then add the following code:

```vue
<script setup lang="ts">
const props = defineProps({
  bvid: String,
});

const getUrl = () => {
  return (
    "//player.bilibili.com/player.html?isOutside=true&autoplay=0&bvid=" +
    props.bvid
  );
};
</script>

<template>
  <div class="w-full h-96 mt-5">
    <iframe
      :src="getUrl()"
      scrolling="no"
      border="0"
      frameborder="no"
      framespacing="0"
      allowfullscreen="true"
      width="100%"
      height="100%"
    >
    </iframe>
  </div>
</template>
```

Then, import and use this component in any blog page where you want to embed a Bilibili video:

```vue
<script setup>
import BiliVideo from "../components/BiliVideo.vue";
</script>
<BiliVideo bvid="BV1q3411b7ps" />
```

Just fill in the corresponding `bvid` value, and you’re good to go!
